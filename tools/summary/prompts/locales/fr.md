# French (fr) — Locale Layer

- **Tone**: Elegant, professional, slightly formal. Avoid being too casual.
- **Register**: "vous" (formal) for prospective students. "tu" only if source establishes it.
- **Text expansion**: 15–25% longer than English.
- **Punctuation**: Space BEFORE colons, semicolons, question marks, exclamation marks ( : ; ? !). Non-breaking space preferred where typography allows.
- **Numbers**: Space for thousands (1 000), comma for decimal (3,5).
- **Currency**: "1 500 € par mois"; "USD 2 000" prefixed when amount is in US dollars.
- **Date**: JJ/MM/AAAA (15/03/2026) or "15 mars 2026".
- **Entity terms**: "séjour linguistique" (language stay), "cours d'anglais" (English course), "école de langue" (language school).
- **Accents**: CRITICAL. Missing accents change meaning (ou/où, a/à, é/è). Always include them.
- **Geography**: "Vancouver, en Colombie-Britannique" on first mention; "la Californie" for California.
- **Hyphenation**: French compound terms use hyphens ("cours d'anglais" not "cours anglais" when meaning "English course").
- **Anti-patterns**: Anglicisms; missing accents; informal "tu" in landing-page contexts.
