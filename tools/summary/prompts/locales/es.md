# Spanish (es) — Locale Layer

- **Tone**: Warm, engaging, conversational but respectful.
- **Regional default**: International Spanish, leaning slightly toward Latin American conventions (CEL's audience includes both Spain and LATAM). Use "tú" for the student-facing voice unless the source uses "usted".
- **Text expansion**: 15–25% longer than English.
- **Punctuation**: Inverted marks at start of questions and exclamations (¿Qué cursos ofrece CEL? / ¡Bienvenido!).
- **Numbers**: Period for thousands (1.000), comma for decimal (3,5). LATAM variant.
- **Currency**: "USD 1.500 al mes" with USD prefix; never assume the user is in any specific country.
- **Date**: DD/MM/AAAA (15/03/2026) or "15 de marzo de 2026".
- **Entity terms**: "curso de inglés" (English course), "escuela de inglés" (English school), "alojamiento estudiantil" (student housing), "estadía lingüística" (language stay).
- **Accents and tildes**: Required. "año" not "ano"; "más" not "mas" when meaning "more".
- **Geography**: "Vancouver, Columbia Británica" on first mention; "California" works as-is.
- **Anti-patterns**: Anglicisms ("Curso" works; "course" doesn't); missing tildes; mixing tú/usted within the same passage.
