# Portuguese (pt) — Locale Layer

- **Tone**: Brazil = warm, friendly, enthusiastic. Portugal = more reserved, formal. CEL's primary PT audience is Brazil; default to Brazilian Portuguese (pt-BR) unless the source clearly uses European conventions.
- **Register**: "você" for student-facing copy; never "tu" (it's regional and informal).
- **Text expansion**: 15–25% longer than English.
- **Numbers**: Period for thousands (1.000), comma for decimal (3,5).
- **Currency**: "US$ 1.500 por mês" (pt-BR convention); "1.500 € por mês".
- **Date**: DD/MM/AAAA (15/03/2026) or "15 de março de 2026".
- **Entity terms**: "curso de inglês" (English course), "escola de inglês" (English school), "intercâmbio" (study-abroad / exchange — common pt-BR term for the student's perspective), "acomodação estudantil" (student housing).
- **Accents**: Required. "inglês" not "ingles"; "está" not "esta" when meaning "is".
- **Geography**: "Vancouver, na Colúmbia Britânica" on first mention; "Califórnia".
- **pt-BR vs pt-PT distinction**: "estudante" (BR) vs "estudante / aluno" (PT) — both fine; "trem" (BR) vs "comboio" (PT) for train. Keep pt-BR default.
- **Anti-patterns**: Direct anglicisms (use Portuguese terms); missing accents; mixing tu/você.
