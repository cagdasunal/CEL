# Italian (it) — Locale Layer

- **Tone**: Warm, expressive, professional. Italian web copy welcomes more flair than German.
- **Register**: "Lei" (formal) for prospective students in education contexts; "tu" if source clearly uses it.
- **Text expansion**: 15–25% longer than English.
- **Numbers**: Period for thousands (1.000), comma for decimal (3,5).
- **Currency**: "USD 1.500 al mese"; "1.500 € al mese".
- **Date**: GG/MM/AAAA (15/03/2026) or "15 marzo 2026".
- **Entity terms**: "corso di inglese" (English course), "scuola di inglese" (English school), "soggiorno linguistico" (language stay), "alloggio studentesco" (student housing).
- **Gender agreement**: Adjectives match noun gender. "corso intensivo" but "scuola intensiva".
- **Geography**: "Vancouver, in Columbia Britannica" on first mention; "California" stays.
- **Apostrophes**: "l'inglese", "un'opportunità" — required where standard.
- **Anti-patterns**: Anglicisms; wrong gender agreement; missing apostrophes.
