# English (en) — Locale Layer

- **Tone**: Direct, factual, professional. Action-oriented.
- **Register**: Standard American English unless source clearly uses BrE conventions. Match the source's existing spelling.
- **Numbers**: Comma for thousands (1,000), period for decimal (3.5).
- **Currency**: $ before amount with USD context, or "CAD" suffix for Canadian content.
- **Date**: "March 15, 2026" or "15 March 2026" — match source.
- **SEO**: Primary keyword in first 100 chars of body; numerals over spelled-out numbers; no fluff transitions.
- **Avoid**: Passive voice unless required by the sentence; jargon ("synergy", "leverage"); sentences over 25 words; English idioms that won't translate ("game changer", "level up").
- **Spelling note**: When the source uses "color" (US) match it; "colour" (UK/CA) only if source already uses it.
