# German (de) — Locale Layer

- **Tone**: Precise, structured, trustworthy. German web copy values thoroughness.
- **Register**: "Sie" (formal) for prospective students; never "Du" unless the source already establishes it. Education context defaults to formal.
- **Text expansion**: German runs 20–35% longer than English. Plan for compact word counts.
- **Compounds**: Compound nouns are correct German style. "Sprachaufenthalt" not "Sprach Aufenthalt". Don't break legitimate compounds for "readability".
- **Capitalization**: ALL nouns are capitalized (grammar, not emphasis).
- **Numbers**: Period for thousands (1.000), comma for decimal (3,5).
- **Currency**: "1.500 € pro Monat", "USD 2.000" — euro symbol after, USD prefix on US amounts.
- **Date**: TT.MM.JJJJ (15.03.2026) or "15. März 2026" written out.
- **Entity terms**: "Sprachaufenthalt" (language stay), "Sprachreise" (language trip), "Sprachschule" (language school), "Sprachkurs" (language course). Use the term that matches the source's existing vocabulary.
- **Anti-patterns**: Anglicisms when a clean German term exists ("Course" → "Kurs"); machine-translated CTAs; missing capitalization on nouns.
- **Geography**: "Vancouver, Britisch-Kolumbien" on first mention; "Kalifornien" for California.
